# Expense Tracker

A simple web-based Expense Tracker built using modern web technologies.

## Features

* Add expenses
* Track spending
* Categorize transactions
* View expense history

## Tech Stack

* HTML
* CSS
* JavaScript

## Installation

```bash
git clone <repository-url>
```

Open the project and run it in your browser.
